Repositorio EduTeAmo

Repositorio oficial para instalar el addon EduTeAmo y el skin Estuary Eduteamo.

Descargar repository.eduteamo.zip
Cómo instalar en Kodi:
  1. Descarga el archivo repository.eduteamo.zip arriba.
  2. Abre Kodi y ve a Ajustes -> Add-ons.
  3. Selecciona Instalar desde archivo zip y elige el archivo descargado.
  4. Una vez instalado, selecciona Instalar desde repositorio -> Repositorio EduTeAmo para instalar el addon y la skin.
