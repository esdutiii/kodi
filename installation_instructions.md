# Kodi Installation Guide

This guide details step-by-step how to configure Kodi from scratch, install the repository, install the video add-on, and finally apply the custom skin.

---

## Step 1: Allow Unknown Sources
By default, Kodi blocks the installation of external add-ons. To enable it:

1. Open **Kodi**.
2. Go to **Settings** (gear icon ⚙️ in the upper left corner).
3. Select the **System** panel.
4. In the left menu, scroll down to the **Add-ons** section.
5. On the right, enable the **Unknown sources** option.
6. Read the warning and click **Yes** to confirm the change.

---

## Step 2: Install the Repository (Adding the Source)
Once Kodi is configured, we will proceed to add the web network source and install the repository:

1. Go back to the Kodi home menu and enter **Settings** (gear icon ⚙️).
2. Select **File manager**.
3. Double-click on **Add source**.
4. Click on `<None>` and type exactly the following web address:
   `https://esdutiii.github.io/kodi`
   Then, in the box below, give the source an identifying name (for example: `EduTeAmo`) and click **OK**.
5. Return to the Kodi home menu, enter **Add-ons**, and click the **open box** icon (Add-on browser) in the upper left corner.
6. Choose the option **Install from zip file**.
7. Select the source you just added (`EduTeAmo`) and click on the file **`repository.eduteamo.zip`**. Wait to receive the notification that it has been installed successfully.

---

## Step 3: Install the Video Add-on
With the repository installed, we will proceed to install the video add-on:

1. On the Add-on browser screen, select **Install from repository**.
2. Select **Eduteamo Repo** (or the corresponding name).
3. Enter the **Video add-ons** category.
4. Select the **EduTeAmo** add-on and click **Install**.
5. If it prompts you to install additional dependencies, click **OK**. Wait to receive the installation confirmation notification.

---

## Step 4: Install the Skin (Appearance) from the Repository
The skin will change the visual appearance of Kodi to apply your customization:

1. On the Add-on browser screen, select **Install from repository**.
2. Select **Eduteamo Repo**.
3. Navigate to **Look and feel** -> **Skin**.
4. Select the **Estuary Eduteamo** skin and click **Install**.

---

## Step 5: Activate the new Skin
When the skin installation finishes, Kodi will automatically ask you if you want to switch to the new appearance:

1. In the prompt **"Would you like to switch to this skin?"**, select **Yes**.
2. In the second prompt **"Would you like to keep this change?"**, select **Yes**.

Done! You now have the system configured in Spanish with your repository, add-on, and active visual appearance.
